# Gate Design

| Gate | Signal | Blocks release |
| --- | --- | --- |
| Contract | Response shape vs API contract | Yes |
| Performance | p95 latency and error rate | Yes |
| Accessibility | HTML image alt text and label hints | Warning by default |
| Flake | Test instability over recent history | Quarantine and warning |

The goal is not to replace Playwright, k6, or Pact. It is to demonstrate the
decision layer that many teams miss after collecting raw test output.
